Pandas Tutorial

Pandas is a fast, powerful, flexible, and easy-to-use open-source data analysis and manipulation tool built on top of Python. It provides data structures and operations for manipulating numerical tables and time series.

Open Source Python 3.9+ NumPy Based

Pandas is built on top of NumPy and is designed to integrate well with many other 3rd party libraries in the scientific Python ecosystem, including Matplotlib, SciPy, and scikit-learn.

Quick Start

Start by importing Pandas. The conventional alias is pd.

PYTHON 
import pandas as pd

# Create a simple DataFrame
data = {
    'Name': ['Alice', 'Bob', 'Carol', 'Dave'],
    'Age': [25, 30, 35, 28],
    'City': ['NY', 'LA', 'Chicago', 'Houston'],
    'Salary': [72000, 85000, 91000, 67000]
}

df = pd.DataFrame(data)
print(df)
Output
Name Age City Salary 0 Alice 25 NY 72000 1 Bob 30 LA 85000 2 Carol 35 Chicago 91000 3 Dave 28 Houston 67000
💡

In a Jupyter notebook, you can simply write df without print() to get a nicely-rendered HTML table automatically.

Series

A Series is a one-dimensional labeled array capable of holding any data type. It is the fundamental building block of Pandas.

PYTHON 
# Create a Series from a list
s = pd.Series([10, 20, 30, 40])

# Series with custom index
s2 = pd.Series(
    [1.2, 3.4, 5.6],
    index=['a', 'b', 'c'],
    name='values'
)

# From a dictionary
s3 = pd.Series({'x': 100, 'y': 200, 'z': 300})

# Key attributes
print(s2.dtype)   # float64
print(s2.index)   # Index(['a', 'b', 'c'], dtype='object')
print(s2.values)  # [1.2 3.4 5.6]
print(s2.shape)   # (3,)

Series Operations

PYTHON 
prices = pd.Series([100, 150, 200, 120], index=['AAPL', 'MSFT', 'GOOG', 'AMZN'])

# Vectorized arithmetic
daily_change = prices * 0.02

# Boolean filtering
above_130 = prices[prices > 130]

# Aggregations
print(prices.mean())   # 142.5
print(prices.max())    # 200
print(prices.idxmax()) # 'GOOG'

DataFrame

A DataFrame is a two-dimensional, size-mutable, potentially heterogeneous tabular data structure with labeled axes (rows and columns). Think of it as a spreadsheet or SQL table.

Creating DataFrames

PYTHON 
# From dictionary
df = pd.DataFrame({
    'A': [1, 2, 3],
    'B': [4.0, 5.0, 6.0],
    'C': ['x', 'y', 'z']
})

# From list of dicts
records = [
    {'name': 'Alice', 'score': 88},
    {'name': 'Bob',   'score': 74},
]
df2 = pd.DataFrame(records)

# From NumPy array
import numpy as np
df3 = pd.DataFrame(
    np.random.randn(4, 3),
    columns=['X', 'Y', 'Z']
)

I/O Tools

Pandas supports reading and writing many file formats. The most common is CSV, but it also handles Excel, JSON, SQL, Parquet, HDF5, and more.

Reading Data

PYTHON 
# CSV
df = pd.read_csv('data.csv')
df = pd.read_csv('data.csv', index_col=0, parse_dates=['date'])

# Excel
df = pd.read_excel('data.xlsx', sheet_name='Sheet1')

# JSON
df = pd.read_json('data.json')

# SQL
import sqlite3
conn = sqlite3.connect('mydb.db')
df = pd.read_sql('SELECT * FROM users', conn)

# Parquet (fast columnar format)
df = pd.read_parquet('data.parquet')

Writing Data

PYTHON 
df.to_csv('output.csv', index=False)
df.to_excel('output.xlsx', sheet_name='Results')
df.to_json('output.json', orient='records')
df.to_parquet('output.parquet', compression='snappy')
df.to_sql('table_name', conn, if_exists='replace')

Indexing & Selection

Pandas provides multiple ways to select subsets of data. Understanding the difference between .loc (label-based) and .iloc (integer-position based) is essential.

PYTHON 
# Select a single column → Series
col = df['Name']

# Select multiple columns → DataFrame
subset = df[['Name', 'Age']]

# .loc — label-based
df.loc[0]                  # row with label 0
df.loc[0:2, 'Name':'Age']  # rows 0-2, cols Name to Age
df.loc[df['Age'] > 28]     # boolean mask

# .iloc — integer position
df.iloc[0]           # first row
df.iloc[1:3, 0:2]   # rows 1-2, cols 0-1
df.iloc[-1]          # last row

# Conditions with & | ~
mask = (df['Age'] > 25) & (df['Salary'] > 70000)
filtered = df[mask]

# .query() — string syntax
result = df.query('Age > 25 and City == "NY"')
⚠

When using .loc, the end of a slice is inclusive. With .iloc, it is exclusive — consistent with Python conventions.

Data Cleaning

Real-world data is messy. Pandas has a comprehensive set of tools to deal with missing values, duplicates, and type conversions.

Missing Values

PYTHON 
# Detect missing values
df.isnull()             # boolean DataFrame
df.isnull().sum()       # count per column
df.isnull().sum().sum() # total missing

# Drop rows/cols with missing values
df.dropna()                          # any NaN
df.dropna(subset=['Age', 'Salary'])  # specific cols
df.dropna(thresh=3)                  # keep if ≥3 non-NaN

# Fill missing values
df.fillna(0)                      # fill with 0
df['Age'].fillna(df['Age'].mean()) # fill with mean
df.fillna(method='ffill')          # forward fill
df.fillna(method='bfill')          # backward fill

Duplicates & Types

PYTHON 
# Duplicates
df.duplicated()                # boolean Series
df.drop_duplicates()           # remove all
df.drop_duplicates(subset=['Name'])  # based on col

# Type conversion
df['Age'] = df['Age'].astype(int)
df['date'] = pd.to_datetime(df['date'])
df['price'] = pd.to_numeric(df['price'], errors='coerce')

# Rename columns
df.rename(columns={'old_name': 'new_name'}, inplace=True)
df.columns = df.columns.str.lower().str.replace(' ', '_')

GroupBy

The groupby operation follows a split → apply → combine pattern. You split the data into groups, apply a function, then combine the results.

PYTHON 
# Basic groupby + aggregation
df.groupby('City')['Salary'].mean()

# Multiple aggregations with .agg()
df.groupby('City').agg({
    'Salary': ['mean', 'max', 'count'],
    'Age': 'median'
})

# Group by multiple columns
df.groupby(['City', 'Department'])['Salary'].sum()

# Apply custom function
df.groupby('City')['Salary'].apply(lambda x: x.max() - x.min())

# transform — returns same-shape result
df['salary_rank'] = df.groupby('City')['Salary'].rank(ascending=False)

Merge & Join

Combine DataFrames using merge(), join(), or concat().

PYTHON 
# SQL-style merge
result = pd.merge(df_left, df_right, on='id', how='inner')

# how options: 'inner', 'left', 'right', 'outer'
result = pd.merge(df_left, df_right,
                 left_on='user_id',
                 right_on='id',
                 how='left')

# Concatenate vertically
combined = pd.concat([df1, df2], ignore_index=True)

# Concatenate horizontally
wide = pd.concat([df1, df2], axis=1)

# Join on index
df_left.join(df_right, how='outer', lsuffix='_l', rsuffix='_r')

Reshape

Pivot, melt, and stack operations let you restructure DataFrames between wide and long formats.

PYTHON 
# Pivot table
pivot = df.pivot_table(
    values='Salary',
    index='City',
    columns='Department',
    aggfunc='mean',
    fill_value=0
)

# Melt (wide → long)
melted = pd.melt(df,
    id_vars=['Name'],
    value_vars=['Q1', 'Q2', 'Q3'],
    var_name='Quarter',
    value_name='Revenue'
)

# Stack / unstack
stacked = df.stack()    # cols → row index level
unstacked = df.unstack() # row level → cols

Statistics

Pandas provides a rich set of statistical methods for both Series and DataFrames.

PYTHON 
# Descriptive statistics
df['Age'].mean()
df['Age'].median()
df['Age'].std()
df['Age'].var()
df['Age'].skew()
df['Age'].kurt()
df['Age'].quantile(0.75)

# Correlations
df.corr()              # pairwise Pearson
df.corr(method='spearman')

# Cumulative operations
df['Revenue'].cumsum()
df['Revenue'].cumprod()
df['Revenue'].cummax()

# Rolling window
df['Sales'].rolling(window=7).mean()  # 7-day MA
df['Sales'].ewm(span=12).mean()       # EWM

Visualization

Pandas integrates with Matplotlib to provide quick built-in plotting. For more polished charts, use the output with Seaborn or Plotly.

PYTHON 
import matplotlib.pyplot as plt

# Line plot (default)
df['Salary'].plot()

# Various chart types
df.plot(kind='bar', x='Name', y='Salary', figsize=(10, 6))
df.plot(kind='scatter', x='Age', y='Salary', alpha=0.7)
df['Salary'].plot(kind='hist', bins=20, edgecolor='white')
df['City'].value_counts().plot(kind='pie')
df.plot(kind='box')   # boxplot

# Correlation heatmap (needs Seaborn)
import seaborn as sns
sns.heatmap(df.corr(), annot=True, cmap='coolwarm')
plt.show()

Time Series

Pandas has extensive support for working with dates, times, and time-indexed data — a core strength of the library.

PYTHON 
# Parse dates
df['date'] = pd.to_datetime(df['date'])

# Date range generation
dates = pd.date_range('2024-01-01', periods=365, freq='D')
biz_days = pd.date_range('2024-01-01', '2024-12-31', freq='B')

# Set date as index
df = df.set_index('date')

# Slice by date
df['2024']
df['2024-01':'2024-06']

# Resample (like groupby for time)
monthly = df['Revenue'].resample('ME').sum()
quarterly = df['Revenue'].resample('QE').mean()

# Timezone handling
df = df.tz_localize('UTC')
df = df.tz_convert('US/Eastern')
